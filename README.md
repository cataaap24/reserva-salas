# Sistema de Reservas de Salas en una Universidad
El proyecto modela un sistema sencillo para reservar salas en una universidad.
## El sistema permite:
- Registrar salas.
- Crear reservas.
- Consultar reservas.
- Cancelar reservas.
- Listar salas y reservas.
- Generar un reporte general.

                                                                                          
  *<ins>En este primer incremento el sistema únicamente permite registrar salas</ins>*
## Tecnologías:
- Java.
- JavaFX.
- Maven.
--- 
## Ejecución
Para ejecutar este proyecto de JavaFX y Maven, puedes descargar el repositorio en formato .zip y seguir estos pasos:
### 1. Requisitos Previos
- Java JDK: Asegúrate de tener instalado Java (versión 11 o superior).
### 2. Preparación
1. Descarga el archivo .zip del repositorio y extráelo en tu computadora.
2. Abre tu Entorno de Desarrollo (IntelliJ IDEA, Eclipse o VS Code).
3. Selecciona la opción Abrir Proyecto (Open Project) y elige la carpeta raíz que contiene el archivo pom.xml.
4. Espera unos instantes a que el IDE reconozca el proyecto y descargue de forma automática todas las dependencias de Maven.
### 3. Comandos de Ejecución
Abre la terminal integrada en la raíz del proyecto y ejecuta el siguiente comando
```bash
mvn javafx:run
```
